Creating a simple material in
Substance Designer

Unreal Engine 4 adopted a physical based material and shading model which is based on the ’Metallic/Specular’ model. To start a newSubstance create choose File -> New Substance and select ’PhysicalBase (Metallic/Roughness)’ and Graph Name: Tiles.

You will start with five tiles: Start Tiles

  • Base Color
  • Normal
  • Roughness
  • Metallic
  • Height

These are the outputs and each represent an image of your material.

Tile Generator Add two 'Uniform Color' nodes and one 'Tile Generator' node (Grayscale). Change to color of the 'Uniform Color' nodes to blue-ish white (#ffd6f2f6) and light blue (#ff49beec) and change the 'Tile Generator' as follows:

  • Scale: 1.1 (this will make the grout gaps very thin)
  • Random Mask: 0.13 (this will render certain tiles black)

Add a Blend node and connect everything up as shown below:

Grout Add a 'Fractal Sum 1' node and attach it to a Gradient node to convert it from grayscale to color. Next add a 'Tile generator' node, set scale to 0.93 to make the lines thicker and invert it. Blend the two together.

Base Color Take to output of the first Blend node and attach it to the middle attachment point of the grout blend node. And attach the grout blend node to the base color.

Roughness Connect the 'Invert Greyscale' node directly to the Roughness output node

Normal Connect the 'Invert Grayscale' to a Normal node and then to the normal output.

Metallic Connect the grout 'Tile Generator' node to a HSL node. In the HSL node lower the lightness from 0.5 to 0.07. Then connect it to the Metallic output.

Height Connect the grout 'Tile Generator' node directly to the Height output node. To be able to see it go the menu of the 3d viewer. Select: Materials -> Default -> Edit. Change te Scale property to 0.02.
